# Python 개념

## 예상 토픽
- 데이터 모델 (`__init__`, `__repr__`, dunder)
- `dataclass` vs 일반 class vs `NamedTuple`
- 타입 힌트 (`typing`, `Optional`, `Literal`, generic)
- 제너레이터 (`yield`, lazy 평가, 메모리 이점)
- 데코레이터 (함수/클래스 데코레이터, `functools.wraps`)
- 컨텍스트 매니저 (`with`, `__enter__/__exit__`)
- 모듈 시스템 (`__main__`, `python -m`, package layout)
- 파일 I/O 패턴 (JSONL, CSV, atomic write via rename)
- 표준 라이브러리 (`pathlib`, `argparse`, `csv`, `json`, `dataclasses`)
- 예외 처리 패턴 (custom exception, exit code)
