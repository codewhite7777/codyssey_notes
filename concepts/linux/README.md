# Linux 개념

## 예상 토픽 (과제별 추출 예정)
- 사용자/그룹/권한 (`chmod`, `chown`, `umask`)
- ACL (`getfacl`, `setfacl`)
- SSH 보안 (포트 변경, PermitRootLogin, key 기반 인증)
- 방화벽 (`ufw` vs `firewalld`)
- cron / crontab / systemd timer
- 프로세스 관리 (`ps`, `top`, `htop`, signals)
- 로그 관리 (`logrotate`, `journalctl`)
- 환경 변수 (`export`, `.bashrc`, `.bash_profile`)
- Bash 스크립트 패턴 (exit code, trap, set -euo pipefail)
