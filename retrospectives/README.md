# 회고 노트

주차별 또는 과제 단위로 학습을 회고.

## 파일명 규칙
`YYYY-MM-DD-<주제>.md`

예시: `2026-05-10-b1-1-monitor-script.md`

## 양식

```markdown
# YYYY-MM-DD <주제>

## 오늘 배운 것
- 핵심 개념 1
- 핵심 개념 2

## 막혔던 지점
- 무엇이 어려웠는가
- 왜 어려웠는가

## 해결 과정
- 시도한 것 / 안 된 이유 / 결국 통한 방법

## 잘못 알고 있던 것
- 교정된 이해

## 다음에 적용할 것
- 인사이트 / 다음 과제에서 어떻게 활용할지

## 추출한 개념
- [개념명](../concepts/<분야>/<주제>.md)
```

`/recap` 슬래시 명령으로 자동 초안 생성 가능.
