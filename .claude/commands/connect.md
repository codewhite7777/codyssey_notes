현재 진행 중인 작업·질문과 `concepts/` 폴더의 기존 학습 노트 사이 연결점을 찾는다.

## 수행

1. `concepts/` 전체 스캔 (분야별 README + 개별 노트 파일)
2. 현재 컨텍스트의 핵심 개념·키워드 추출
3. 연결되는 노트 매칭 → **어떻게 연결되는지** 1-2줄 설명
4. 누락된 개념 식별 → "이건 새로 정리할 가치 있음" 제안

## 출력 양식

```
## 직접 연결되는 기존 노트
- ✅ `concepts/linux/permissions.md` — 이번 ACL 설정에서 권한 모델 활용
- ✅ `concepts/os/signals.md` — Watchdog의 SIGTERM 동작 이해 기반

## 강한 관련성 (참고 권장)
- 🔗 `concepts/linux/cron.md` — 자동화 맥락에서 유사 패턴

## 새로 정리할 가치 있는 개념
- 🆕 cron 환경에서의 PATH 처리 — 자주 함정에 빠짐
- 🆕 monitor.sh 식 스트리밍 로깅 패턴

## 추천 액션
- `/extract`로 새 개념 정리
- 또는 `/learn <주제>`로 미지의 개념 먼저 학습
```

## 원칙

- 억지 연결 X — 진짜 의미 있는 것만
- 연결 설명에 "왜 관련 있는지" 한 줄 필수
- 학생이 "아 이게 그거구나" 깨닫게 하는 게 목표
